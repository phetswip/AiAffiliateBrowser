# AiAffiliate Browser

Production Android browser built with WebView engine.

## Features
- 🖥️ Desktop Chrome mode (default)
- 🎨 Liquid Glass dark UI
- 🧩 Extension Manager
- 📦 Ai Affiliate Academy pre-installed
- ⬇️ Download Manager
- 🔄 Pull-to-Refresh
- 🇹🇭 Thai language support

## Build
```bash
./gradlew assembleRelease
```

Or push to GitHub to trigger the GitHub Actions CI build.
